# KIAN · 具身智能公益学习平台

> Learn · Build · Share｜让机器人学习不再有门槛

KIAN 是一个面向具身智能与机器人学习者的中文公益学习平台，由上海大学张鹏宇发起。平台将学习路线、经典与前沿论文、论文精读、优秀开源项目和共创者展示集中在一个网站中，所有公开学习内容均可免费浏览。

## 投稿与上传入口

### 分享优秀开源项目

投稿地址：[https://kian-embodied-ai.pengyu9711791686.chatgpt.site/projects#open-projects](https://kian-embodied-ai.pengyu9711791686.chatgpt.site/projects#open-projects)

投稿人可以填写项目名称、项目链接、创新点、介绍，以及自愿公开的姓名、学校和微信号。提交后内容会进入待审核状态，管理员审核通过后才会在公开页面展示。

### 发布论文精读

目前论文精读采用管理员统一整理和发布的方式。准备内容时请包含：

- 论文标题与论文原文链接
- 主要创新点
- 核心方法
- 实验结果
- 可以实现或复现的途径
- “一张图了解这篇文章”的配图

普通访问者不能直接发布或删除论文精读。需要投稿时，可联系平台发起者：微信 `kian060701`，或抖音 `kian0711`。

## 内容上线流程

```text
社区投稿 → 进入待审核列表 → 管理员核对内容和链接 → 审核通过 → 公开展示
                                      └→ 不合适或失效内容：删除/暂不上线
```

管理员审核时，请重点检查项目或论文链接是否可以打开、内容是否与机器人或具身智能相关、介绍是否真实清楚，以及投稿人公开信息是否经过本人同意。
